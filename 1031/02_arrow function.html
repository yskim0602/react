02_화살표 함수(arrow function)

        1. 기존함수
        function 함수명(){
            실행할 내용;
        }
        2. 변수선언식 
        let 변수명 = function(){
            실행할 내용;
        }
        함수명();

        3.화살표 함수
        let 함수명 = () => {  //function 대신에 => 기호를 사용한다.
            실행할 내용;
        }

this : 전역에서 this키워드를 작성하면 브라우저 전역을 의미하는 [window]객체입니다.

함수안에서 this키워드는 마찬가지로 [window]객체를 말한다.

객체안에서 사용한 this는 메서드를 소유한 객체를 의미한다.(person)
